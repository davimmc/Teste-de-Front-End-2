Primeiramente agradeço a oportunidade de apresentar minhas capacidades
através desta demonstração realizada sem nenhum framework CSS ou códigos prontos
de javascript.

Primando pela lei 9 de Napoleon Hill "Hábito de fazer mais do que a obrigação",
quero apresentar um pouco mais do que sou capaz de fazer.


Essa apresentação utiliza-se de:

 - Servidor local;
 - HTML5, CSS3, Javascript, JQuery, Ajax, PHP -> PDO;

# Requisitos de funcionamento

  1 - Para que esse projeto funcione corretamente deve-se descompactar o arquivo
  dentro da RAIZ -> htdocs ou www depende  do Servidor.

  2 - Faça a importação no phpmyadmin do arquivo dentro da pasta do projeto
  chamado "desafio.sql" para criar o Database e as Tabelas.

  3 - Verifique o usuário e senha do banco no arquivo /desafio/assets/conexao/bd-dados.php
      - Coloque o IP do PC na variável: SERVER_NAME do arquivo bd-dados.php
      - Verifique se está conectado a Base acessando o Segundo Desafio com a
        "Listando os contatos"

  4 – Acesso o desafio -> http://IPLOCAL/desafio
